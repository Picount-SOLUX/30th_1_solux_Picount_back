# PICOUNT

2025 SOLUX 30기 1학기 프로젝트


## **💻** 서비스 소개

### PICOUNT : Piece + Count

예산을 ‘**조각내어(Piece)**’ ‘**관리하는(Count)**’ 가계부 웹 서비스


## **📍 기획 의도**

**PICOUNT는 예산을 카테고리별로 조각내어 시각적으로 관리하고, 감성적인 기록 경험을 통해 소비 습관을 지속할 수 있도록 돕는 웹 기반 가계부입니다. 가계부의 진입장벽을 낮추고, 꾸미는 재미로 사용자가 자발적으로 소비를 돌아보게 만드는 것이 핵심 목표입니다.**
